## ✨ Features

- 🧠 **AI Analysis** — Paste any startup idea and get a full viability report instantly
- 📊 **5 Metrics** — Market size, Problem clarity, Competitive edge, Revenue potential, Execution difficulty
- 💪 **Strengths & Risks** — Know what works and what could fail
- 💡 **Opportunities** — Uncover what you might be missing
- 🎯 **Recommendation** — Actionable next steps to move forward
- 👤 **User Accounts** — Sign up, sign in, sessions saved locally
- 🗂️ **Dashboard** — All past analyses saved and expandable
- 🔗 **Share Results** — Copy a formatted report to clipboard
- ⚡ **Zero dependencies** — Pure HTML, CSS, JS. No framework, no build step
